# react-apollo-auth

```sh
git clone https://github.com/ctbradley/react-apollo-auth.git
cd react-apollo-auth
cp .env.example .env
```

-   Setup database on mlab.com
-   Edit **.env**
-   Enter **MONGO_DB_HOST, MONGO_DB_USER, MONGO_DB_PASS**

## Project setup

```sh
yarn install
```

## Start server @ localhost:4000/#/

```sh
yarn dev
```

## Lints and fixes files

```sh
yarn lint
```
